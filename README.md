# TaskFlow

TaskFlow is a modern and aesthetic task management web application built using React and Vite.

## Features

* Task scheduling with alarms
* Browser notifications
* Dark mode support
* Edit and delete tasks
* Task categories
* Local storage persistence
* Responsive glassmorphism UI

## Tech Stack

* React
* Vite
* JavaScript
* CSS3

## Run Locally

```bash
npm install
npm run dev
```
